HOW TO PLAY

Players must dodge the red projectiles and collect the white projects to reedem health points and clear the area. The aim of the game is to get the longest survival time possible - this especially becomes competitive when playing against multiple users. The time is displayed in the background. Player controls for each coloured player are listed below the game window. The hostile projectile count is shown on the lower end the game window; the first value represents the current number of projectiles, white the second value shows the cap - this increases over time, making the game become challenging as the game progresses.

Please use Google Chrome. Microsoft Edge is currently unsupported.
